## NOTICE
 - 要先运行`swag init`
 - `@Tags accounts`，相当于分组，表示api就在accounts这个分组下
 - `@Router /testapi/get-string-by-int/{some_id} [get]`，这个url要加上`@BasePath /api/v1`，才是最终的url
 - 可以在启动的时候指定doc.json的位置，在disablingWrapHandler里面用的是默认的配置，需要修改的话要用disablingCustom
